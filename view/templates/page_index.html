G-NAF LDAPI

This is a test Linked Data API delivering data from the G-NAF, the Geocoded National Address File.

Starting points

  • Address Register - the list of all Addresses, paged
  • An Address - example, GAACT714845933
  • Street Locality Register - the list of all Addresses, paged
  • A Street Locality - example, NSW2856338
  • SPARQL endpoint
